# 🩺 دليل تشغيل منصة الدفعة الطبية الموحدة
> **خاص بالأدمن: أبانوب سامح (Intensive Nursing)**

أهلاً بك يا أدمن! هذا المجلد يحتوي على كامل الهيكل البرمجي المحدث والذكي الخاص بمنصة الدفعة لبرنامج التمريض المكثف (6 ترمات). تم إلغاء الأدوات المعقدة، وبناء **لوحة تحكم إدارية (Admin Dashboard)** كاملة وسهلة الاستخدام تتيح لك إدارة المواد، المحاضرات، بنوك الأسئلة، والملفات مباشرة من متصفح الويب الخاص بك وتحديث الموقع فوراً.

---

## 📂 هيكل المجلدات الجديد (Folder Structure)

* 📄 `index.html`: الصفحة الرئيسية للمنصة (التي يفتحها زملائك على الموبايل والكمبيوتر).
* 📄 `admin.html`: لوحة التحكم الإدارية الخاصة بك لإضافة، تعديل، وحذف المحاضرات.
* 📄 `server.py`: خادم بايثون خفيف يعمل محلياً لاستقبال الملفات وتحديث قاعدة البيانات.
* 📄 `run_admin.bat`: زر التشغيل السريع (دبل كليك) لتشغيل لوحة التحكم فوراً.
* 📄 `curriculum.js`: قاعدة بيانات المحاضرات والمواد في المنصة (يتم تحديثها تلقائياً).
* 📁 `assets/`: ملفات التصميم والأكواد الخاصة بالمنصة (`portal.css` و `portal.js`).
* 📁 `terms/`: مجلد يحتوي على الملفات المرفوعة مقسمة حسب الترم والمادة كالتالي:
  - `original/`: الباوربوينت الأصلي للدكتورة.
  - `guides/`: الباوربوينت المترجم أو دليل الشرح.
  - `quizzes/`: بنوك الأسئلة (سواء الخاصة بك أو الخاصة بالدكتورة).

## ⚙️ 1. طريقة إضافة وتعديل المحاضرات (محلياً)

لتشغيل لوحة الأدمن والبدء في رفع الملفات وتحديث المنصة:

1. اضغط مرتين (Double Click) على ملف **`run_admin.bat`**.
2. ستفتح لك شاشة سوداء صغيرة (الخادم)، وسيفتح متصفح الويب تلقائياً على الرابط: `http://localhost:8000/admin.html`.
3. **لإضافة محاضرة جديدة:**
   - اختر الترم الدراسي (من 1 إلى 6).
   - اختر المادة (أو اضغط زر ➕ لإضافة مادة جديدة تماماً للترم النشط).
   - أدخل رقم المحاضرة وعنوانها بالعربي.
   - ارفع الملفات المتاحة لديك في الخانات المخصصة (الباور الأصلي، الباور المترجم، بنك الأسئلة الخاص بك، أو بنك أسئلة الدكتورة).
   - اضغط **"حفظ وإضافة المحاضرة"**.
4. **لتعديل محاضرة (مثلاً لرفع ملف كان ناقصاً):**
   - اختر المادة من القائمة، وستظهر كل محاضراتها في جدول يسار الشاشة.
   - اضغط على زر **✏️ تعديل** بجانب المحاضرة المطلوبة.
   - ستُحمّل بياناتها في النموذج، يمكنك رفع الملف الجديد أو تعديل الاسم، ثم اضغط **"حفظ التعديلات"**.
5. **لحذف محاضرة:**
   - اضغط على زر **❌ حذف** بجانب المحاضرة في الجدول لتُمسح تماماً من قاعدة البيانات.

## 🌐 2. طريقة رفع وتحديث الموقع للدفعة مجاناً 100%

بعد إجراء تعديلاتك محلياً والتأكد من ظهورها في موقع الطلاب عبر زر **"فتح موقع الطلاب"**، اتبع أحد الخيارين لمشاركة التحديثات أونلاين:

### الخيار الأول: Netlify Drop (الأسهل والأسرع)
1. افتح متصفحك واذهب إلى: [Netlify Drop](https://app.netlify.com/drop) (لا تحتاج حتى لإنشاء حساب!).
2. اسحب مجلد **`Nursing_Portal`** بالكامل وأفلته داخل الصفحة.
3. خلال ثوانٍ، سيتم رفع الموقع ويعطيك رابطاً عاماً ومجانياً تماماً (مثال: `https://nursing-batch.netlify.app`).
4. انسخ الرابط وشاركه مع زملائك على جروب الدفعة.
5. **عند إضافة محاضرة جديدة مستقبلاً:** قم فقط بسحب وإفلات المجلد مجدداً في نفس الصفحة على Netlify Drop، وسيتم تحديث الرابط العام فوراً عند جميع الطلاب.

### الخيار الثاني: GitHub Pages (للتحديثات الدائمة بنقرة زر)
إذا كنت تفضل استخدام Git والمستودعات:
1. قم بإنشاء مستودع (Repository) جديد على GitHub واجعله Public.
2. ارفع مجلد `Nursing_Portal` عليه.
3. اذهب إلى إعدادات المستودع (Settings) -> Pages.
4. اختر فرع الـ `main` كـ Source واضغط حفظ (Save).
5. ستحصل على رابط ثابت ومجاني للأبد (مثال: `https://abanoub-sameh.github.io/Nursing_Portal/`).
6. **المميز هنا:** بمجرد عمل `git push` بعد إضافة أو تعديل أي محاضرة في المجلد المحلي، سيتحدث الموقع أونلاين تلقائياً وبدون أي تدخل إضافي!
